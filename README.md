
## CRUD BOOK

CRUD simples de livros em Laravel + tailwindcss (incluíndo plugin do formulário).

- Configurar .ENV para conectar no banco
- Importar a tabela de livros, no arquivo .SQL
- Instalar as dependencias com composer e npm
